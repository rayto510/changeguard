# Roadmap

## v0.1 (Current - Dec 2025)
**Foundation & MVP**
- ✅ Core backend APIs (CRUD for schema changes)
- ✅ Basic frontend UI for change management
- ✅ User authentication with JWT
- ✅ PostgreSQL + Redis infrastructure
- ✅ Docker containerization
- In Progress: Basic commenting system

## v0.2 (Q1 2026)
**Core Features**
- Real-time updates with WebSockets
- Status workflow management (open → resolved → archived)
- Service impact analysis
- Email notifications
- Change history and audit logs

## v1.0 (Q2 2026)
**Production Ready**
- Role-based access control (RBAC)
- Team management
- Advanced filtering and search
- Analytics dashboard
- Change statistics and trends
- API documentation

## v2.0 (Q4 2026)
**Enterprise & Scale**
- Multi-tenant support
- Advanced permission model
- SSO (SAML, OIDC)
- Compliance reports (SOC2, GDPR)
- Custom webhooks
- API versioning management
